# TypeScript 笔记
## 配置git
* 设置全局用户名和邮箱
```bash
git config --global user.name "Your Name"
git config --global user.email "your email"
```
## 查看仓库状态
```bash
git status
``` 
## 提交
```bash
git add 文件
git commit -m "message"
``` 
## 对比更改
使用`git diff`比较更改
```bash
git diff 文件
```  
## 回退
1. 使用`git log`查看提交记录
```bash
git log
```
